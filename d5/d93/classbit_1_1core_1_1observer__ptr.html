Detailed Description

template<typename T>
class bit::core::observer_ptr< T >

A lightweight wrapper around a pointer type.

This is essentially a 'dumb' smart pointer, in that it behaves identically to a raw pointer, but exposes a smart-pointer API with raw-pointer semantics.
