39 /*
40  * Symbols Defined:
41  *
42  * BIT_STRINGIZE(a) : Converts symbol 'a' to string at compile time.
43  * BIT_JOIN(lhs,rhs) : concatenates lhs to rhs at compile time, producing symbol lhsrhs.
44  * BIT_TO_BOOL(x) : Converts integer x to a boolean value
45  * BIT_BOOL_NOT(x) : Inverts the boolean value (1 to 0 or 0 to 1)
46  * BIT_BOOL_OR(x,y) : Performs bitwise or, returning the result of x||y
47  * BIT_BOOL_AND(x,y) : Performs bitwise and, returning the result of x&&y
48  * BIT_BOOL_XOR(x,y) : Performs bitwise xor, returning the result of x^y
49  * BIT_BOOL_NAND(x,y) : Performs bitwise nand, returning the result of !(x&&y)
50  * BIT_BOOL_NOR(x,y) : Performs bitwise nor, returning the result of !(x||y)
51  * BIT_BOOL_XNOR(x,y) : Performs bitwise xnor, returning the result of !(x^y)
52  * BIT_IF(cond,x) : If condition is true, perform x
53  * BIT_IF_ELSE(cond,x,y) : If condition is true, perform x, otherwise y
54  * BIT_COMMA_IF(cond) : If condition, create comma
55  * BIT_LIST(c,p) : Creates an increasing list of c elements, prefixed by 'p'
56  * BIT_REVERSE_LIST(c,p) : Creates a decreasing list of c elements, prefixed by 'p'
57  * BIT_SELECT(n,...) : Selects the n'th argument of the supplied variadic list
58  * BIT_VA_ARGS(...) : Portable way of passing variadic arguments without a trailing comma on empty input
59  * BIT_EMPTY : Empty symbol for passing to macro parameters to avoid warnings
60  */
