Detailed Description

Inline namespace for managing cast utilities.

This allows for better granularity, so that simple casts can be imported and used globally

To bit::core::casts::copy_cast ( const From &  from)
noexcept

Performs a cast between types by copying bytes from from to the new type.

Note
Bytes are truncated to fit into type To. If the size of From is smaller than To, the remaining bytes are left untouched (and thus may be UB to access)
This cast does not participate in overload resolution unless is_trivially_copyable<To>::value and is_trivially_copyable<From>::value are both true

Performs an implicit cast between two types, such that the cast is actually performed using the implicit language casting.

This can be used to signal cases that would otherwise produce warnings when type promotions are performed.

Note
This cast does not participate in overload resolution unless is_implicitly_convertible<From,To>::value is true

Performs a bounded casting safely between numeric types.

This method will assert if the value being casted does not properly fit into the value being converted to. Calling this method has the same form as calling static_cast

Note
