Detailed Description

template<typename T>
class bit::core::offset_ptr< T >

An offset pointer based on boost::offset_ptr.

Note
Like boost::offset_ptr, an offset value of '1' is used to represent nullptr internally, since it is unlikely for an offset of 1 to ever be valid. 0 is not used, since a self-assignment of an offset_ptr is a valid use-case. Since 8-bit pointers are exceedingly rare, it should be unlikely to ever have an issue with this. An offset of 1 would be pointing into the offset_ptr class.
