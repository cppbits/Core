Detailed Description

template<typename... InputIterators>
class bit::core::zip_iterator< InputIterators >

An iterator used for zipping multiple iterator ranges together into a single iterator.

A zip_iterator does not have any operator->, since the underlying reference type is a std::tuple<T&,...> for each iterator entry.
