Detailed Description

template<typename T>
class bit::core::array_view< T >

An immutable wrapper around non-owned contiguous data.

This type behaves like a const span, but is only constructible from contiguous data containers and raw arrays, not from pointer + size.

Satisfies ContiguousContainer ContiguousContainer
